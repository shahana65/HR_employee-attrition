# HR Analytics – Employee Attrition Analysis 📊

## Overview

This project focuses on analyzing employee attrition data using Python. The goal is to identify patterns and factors that may influence employees leaving a company.

Using data analysis and visualization techniques, the project explores employee-related information such as age, salary, department, job role, and work experience.

---

## Technologies & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## Dataset

The dataset used in this project is the IBM HR Employee Attrition dataset.

It contains employee-related details such as:

* Age
* Department
* Job Role
* Monthly Income
* Work Experience
* Attrition Status
* Job Satisfaction
* Work-Life Balance
* And more

---

## Project Workflow

### 1. Data Loading

* Imported the dataset using Pandas
* Explored the dataset structure
* Checked for missing values

### 2. Data Preparation

* Converted Attrition values:

  * Yes → 1
  * No → 0
* Cleaned and prepared data for analysis

### 3. Data Analysis

Performed analysis on:

* Average employee age
* Average monthly income
* Attrition by department
* Salary comparison by job role
* Employee attrition trends

### 4. Data Visualization

Created visualizations using Matplotlib and Seaborn:

* Countplot for attrition
* Histogram for age distribution
* Boxplot for salary vs attrition
* Correlation heatmap

---

## Key Insights

* Sales department showed a higher attrition rate.
* Managers and Research Directors had the highest average income.
* Employees with lower salaries showed higher attrition tendencies.
* Most employees were between 25–45 years old.
* Multiple factors such as overtime, income, and job satisfaction influenced attrition.
